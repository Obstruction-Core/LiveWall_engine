## вњЁ Features

### рџЋ¬ Professional-Grade Preview
* **Native HW Acceleration** вЂ“ Real-time playback for `MP4`, `MKV`, and `WebM` directly within the GUI using `QtMultimedia`.
* **Dynamic GIF Rendering** вЂ“ Smooth, infinite looping via `QMovie` for animated wallpapers.
* **Aspect-Aware Scaling** вЂ“ Intelligent UI that scales previews without distorting the original media's ratio.

### рџЋЁ Advanced Matugen Logic
* **9 Specialized Schemes** вЂ“ Full access to `scheme-content`, `expressive`, `fidelity`, `fruit-salad`, `monochrome`, `neutral`, `rainbow`, `tonal-spot`, and `vibrant`.
* **Palette Inspector** вЂ“ Visual swatches show the exact HEX values generated by the algorithm.
* **Instant Clipboard** вЂ“ Click any swatch to instantly copy the HEX code for manual CSS/Config tweaks.

### рџ”„ Atomic State & Safety
* **Pre-Flight Backups** вЂ“ Automatically snapshots `config.json` before every theme application.
* **One-Click Rollback** вЂ“ Instantly restores the previous wallpaper, Light/Dark mode, and color scheme if the result isn't perfect.
